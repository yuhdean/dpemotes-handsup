# dpemotes-handsup (fork)
Adds keybinding for the handsup emote, toggled via X.
Adds emote cancellation for when a firearm is pulled out.
Adds weapon dropping & picking up when a firearm is held at the time of putting your hands up. 

## Dependencies
- [ox_lib](https://github.com/overextended/ox_lib)
- [ox_target](https://github.com/overextended/ox_target)

# dpemotes (original)
Emotes / Animations for fiveM with Prop support.

Installation Instructions:

add dpemotes to your server.cfg

start dpemotes

Other instructions please check the fivem forum thread
